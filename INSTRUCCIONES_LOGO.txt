✅ PASO 1: Preparar tus imágenes
────────────────────────────────────────────────────────────────────────
Necesitas 2 archivos:

📄 Favicon (ícono de la pestaña del navegador):
   - Nombre: favicon.ico
   - Tamaño recomendado: 32x32 píxeles
   - Formato: .ico (preferido) o .png

📄 Logo (para login y dashboard):
   - Nombre: logo.png
   - Tamaño recomendado: 200x200 píxeles o más
   - Formato: .png con fondo transparente (recomendado)


✅ PASO 2: Copiar archivos a la carpeta public
────────────────────────────────────────────────────────────────────────
La carpeta ya está creada aquí:
📁 C:\Users\LENOVO\Documents\projects\warehouse-stocks\public\

OPCIÓN A - Usando el Explorador de Windows:
1. Abre el Explorador de Windows
2. Navega a la carpeta de tus imágenes
3. Copia tus 2 archivos
4. Pega en: C:\Users\LENOVO\Documents\projects\warehouse-stocks\public\
5. Renombra si es necesario a:
   → favicon.ico
   → logo.png

OPCIÓN B - Usando PowerShell/CMD:
Abre PowerShell en la ubicación de tus imágenes y ejecuta:

copy tu-favicon.ico C:\Users\LENOVO\Documents\projects\warehouse-stocks\public\favicon.ico
copy tu-logo.png C:\Users\LENOVO\Documents\projects\warehouse-stocks\public\logo.png


✅ PASO 3: Verificar que los archivos estén correctos
────────────────────────────────────────────────────────────────────────
Abre PowerShell en la carpeta del proyecto y ejecuta:

dir public

Deberías ver:
favicon.ico
logo.png


✅ PASO 4: Reiniciar el servidor
────────────────────────────────────────────────────────────────────────
1. En la terminal donde está corriendo npm run dev:
   - Presiona: Ctrl + C
   - Espera que se detenga
   
2. Vuelve a iniciar:
   npm run dev


✅ PASO 5: Ver los cambios
────────────────────────────────────────────────────────────────────────
1. Abre tu navegador
2. Ve a: http://localhost:3000/login
3. Deberías ver:
   ✓ Tu logo en la página de login
   ✓ Tu favicon en la pestaña del navegador

4. Inicia sesión y verifica:
   ✓ Tu logo en el sidebar del dashboard
   ✓ Cuando colapses el sidebar, verás el logo más pequeño

❌ Error: "Failed to load image"
   → Verifica que los archivos se llamen exactamente:
     • favicon.ico
     • logo.png
   → Verifica que estén en la carpeta public/
   → Reinicia el servidor (npm run dev)

❌ El logo se ve distorsionado:
   → Usa una imagen cuadrada (1:1)
   → Tamaño mínimo recomendado: 200x200 px
   → Formato PNG con fondo transparente

❌ El favicon no aparece:
   → Limpia la caché del navegador
   → Cierra completamente el navegador y vuelve a abrir
   → Verifica que el archivo sea favicon.ico
